# Personal Expense Tracker

This is a straightforward app built in Java to help you keep an eye on your daily spending. You can add expenses as you go, look them up later, and break things down by categories like groceries or entertainment. It’ll tally up everything for you—totals by category and overall—so you always know where your money’s going. All your data sticks around, thanks to file storage; next time you launch the app, your expenses are still there. The whole thing relies on OOP principles and uses Java collections to keep everything organized.

# What it does:
- Lets you add expenses easily
- Shows all your expenses in one place
- Figures out the total you’ve spent
- Gives you a summary for each category
- Saves your data to a file, so nothing gets lost

# How to use it:
- Compile: javac *.java
- Run: java Main

# Tech behind it:
- Java
- OOP concepts
- File handling
