# personal-expense-tracker
A simple Java-based application to track and manage daily expenses. Supports adding, viewing, and analyzing expenses by category, with total calculations and file handling for data persistence. Built using OOP concepts and collections.
## Features
- Add expenses
- View expenses
- Calculate total
- Category-wise summary
- File storage

## How to Run
1. Compile all files:
   javac *.java
2. Run:
   java Main

## Technologies Used
- Java
- OOP Concepts
- File Handling
